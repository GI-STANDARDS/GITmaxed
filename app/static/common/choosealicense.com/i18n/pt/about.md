O GitHub quer ajudar quem desenvolve a escolher uma licença de código aberto para o seu código-fonte.

Se você já sabe o que está fazendo e tem uma licença que prefere usar, ótimo! Não estamos aqui para fazer você mudar de ideia. Mas se o grande número de opções deixa você confuso, talvez possamos ajudar.

## Não é abrangente

Este site não é um diretório abrangente de licenças de código aberto. A grande maioria dos projetos provavelmente será bem atendida por uma das três opções destacadas na [página inicial](/) — escolher a licença [preferida](/community/) por projetos semelhantes ao seu, ou a licença permissiva mais popular ([MIT](/licenses/mit/)), ou a licença copyleft mais popular ([GNU GPLv3](/licenses/gpl-3.0/)). Caso você tenha necessidades específicas não cobertas por essas, também destacamos [algumas outras licenças a considerar](/licenses/) e temos uma página sobre [licenças para projetos que não são software](/non-software/).

Consulte nosso [apêndice](/appendix/) para uma tabela com todas as licenças catalogadas no [repositório do choosealicense.com](https://github.com/github/choosealicense.com) e os links abaixo para *ainda mais* licenças que você **não precisa** conhecer para mesmo assim escolher uma excelente licença para o seu projeto.

### Recursos adicionais

## Ajude-nos a melhorar

O choosealicense.com não trata apenas de código aberto; o próprio site também é de código aberto. Viu algo que acha que poderia ser feito melhor? Fique à vontade para [fazer um fork do projeto](https://github.com/github/choosealicense.com) no GitHub e enviar um pull request. Suas melhorias são bem-vindas.

## Aviso legal

Não somos advogados. Bem, a maioria de nós, pelo menos. O objetivo deste site não é fornecer aconselhamento jurídico. O objetivo deste site é oferecer um ponto de partida para ajudar você a fazer uma escolha consciente, apresentando de forma simplificada informações sobre licenças de código aberto populares. Se você tiver alguma dúvida sobre a melhor licença para o seu código ou qualquer outra questão jurídica relacionada a ele, é sua responsabilidade pesquisar mais a fundo ou consultar um profissional.
