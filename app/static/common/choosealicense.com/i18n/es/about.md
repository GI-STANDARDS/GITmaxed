GitHub quiere ayudar a las personas desarrolladoras a elegir una licencia de código abierto para su código fuente.

Si ya sabes lo que haces y tienes una licencia que prefieres usar, ¡estupendo! No estamos aquí para hacerte cambiar de idea. Pero si el gran número de opciones te abruma, quizá podamos ayudarte.

## No es exhaustivo

Este sitio no es un directorio exhaustivo de las licencias de código abierto. A la gran mayoría de los proyectos probablemente les servirá bien una de las tres opciones destacadas en la [página de inicio](/): elegir la licencia [preferida](/community/) por proyectos similares al tuyo, o la licencia permisiva más popular ([MIT](/licenses/mit/)), o la licencia copyleft más popular ([GNU GPLv3](/licenses/gpl-3.0/)). Por si tienes necesidades concretas no cubiertas por estas, también destacamos [algunas otras licencias que considerar](/licenses/) y tenemos una página sobre [licencias para proyectos no informáticos](/non-software/).

Consulta nuestro [apéndice](/appendix/) para ver una tabla de todas las licencias catalogadas en el [repositorio de choosealicense.com](https://github.com/github/choosealicense.com) y los enlaces de abajo para *aún más* licencias que **no necesitas** conocer para elegir igualmente una licencia excelente para tu proyecto.

## Ayúdanos a mejorarlo

Choosealicense.com no trata solo del código abierto; el propio sitio también es código abierto. ¿Ves algo que crees que podría hacerse mejor? No dudes en [hacer un fork del proyecto](https://github.com/github/choosealicense.com) en GitHub y enviar una pull request. Tus mejoras son bienvenidas.

## Aviso legal

No somos abogados. Bueno, la mayoría de nosotros. El objetivo de este sitio no es ofrecer asesoramiento jurídico. El objetivo de este sitio es ofrecer un punto de partida que te ayude a tomar una decisión informada, presentando de forma simplificada información sobre las licencias de código abierto populares. Si tienes alguna duda sobre cuál es la mejor licencia para tu código, o sobre cualquier otra cuestión jurídica relacionada con él, es tu responsabilidad investigar más o consultar a un profesional.
