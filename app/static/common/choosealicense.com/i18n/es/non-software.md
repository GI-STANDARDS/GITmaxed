Las licencias de software de código abierto también pueden usarse para obras no informáticas y a menudo son la mejor opción, sobre todo cuando las obras en cuestión pueden editarse y versionarse como código fuente. [Elige una licencia de código abierto](/).

### Datos, contenido multimedia, etc.
{: #data-media-etc }

[CC0-1.0](/licenses/cc0-1.0/), [CC-BY-4.0](/licenses/cc-by-4.0/) y [CC-BY-SA-4.0](/licenses/cc-by-sa-4.0/) son licencias [abiertas](https://opendefinition.org) usadas para material no informático que va desde conjuntos de datos hasta vídeos. Ten en cuenta que Creative Commons [no recomienda usar sus licencias para software](https://creativecommons.org/faq/#can-i-apply-a-creative-commons-license-to-software) ni para hardware.

### Documentación

Cualquier licencia de software de código abierto o licencia abierta para contenido multimedia (consulta [más arriba](#data-media-etc)) también es válida para la documentación de software. Si usas licencias distintas para tu software y su documentación, asegúrate de indicar que los ejemplos de código fuente de la documentación también están bajo la licencia del software.

### Tipografías

La [SIL Open Font License 1.1](/licenses/ofl-1.1/) mantiene abiertas las tipografías, permitiendo que se usen libremente en otras obras.

### Hardware

Los diseños de [hardware de código abierto](https://oshwa.org/resources/open-source-hardware-definition/), desde mobiliario hasta [FPGA](https://oshwa.org/resources/best-practices-for-sharing-fpga-designs/), están cubiertos por las licencias CERN Open Hardware: [CERN-OHL-P-2.0](/licenses/cern-ohl-p-2.0/) (permisiva), [CERN-OHL-W-2.0](/licenses/cern-ohl-w-2.0/) (débilmente recíproca) y [CERN-OHL-S-2.0](/licenses/cern-ohl-s-2.0/) (fuertemente recíproca).

### Proyectos mixtos

Si tu proyecto contiene una mezcla de distintos tipos de material, puedes incluir varias licencias, siempre que indiques de forma explícita qué licencia se aplica a cada parte del proyecto. Consulta [el aviso de licencia de este sitio](https://github.com/github/choosealicense.com#license) como ejemplo.
