Les licences de logiciels open source peuvent aussi servir pour des œuvres non logicielles et constituent souvent le meilleur choix, surtout lorsque les œuvres concernées peuvent être éditées et versionnées comme du code source. [Choisissez une licence open source](/).

### Données, médias, etc.
{: #data-media-etc }

[CC0-1.0](/licenses/cc0-1.0/), [CC-BY-4.0](/licenses/cc-by-4.0/) et [CC-BY-SA-4.0](/licenses/cc-by-sa-4.0/) sont des licences [ouvertes](https://opendefinition.org) utilisées pour des contenus non logiciels, des jeux de données aux vidéos. Notez que Creative Commons [ne recommande pas l’usage de ses licences pour les logiciels](https://creativecommons.org/faq/#can-i-apply-a-creative-commons-license-to-software) ni pour le matériel.

### Documentation

Toute licence de logiciel open source ou licence ouverte pour les médias (voir [ci-dessus](#data-media-etc)) s’applique aussi à la documentation logicielle. Si vous utilisez des licences différentes pour votre logiciel et sa documentation, veillez à préciser que les exemples de code source figurant dans la documentation sont également placés sous la licence du logiciel.

### Polices

La [SIL Open Font License 1.1](/licenses/ofl-1.1/) garde les polices ouvertes, en permettant leur libre utilisation dans d’autres œuvres.

### Matériel

Les conceptions de [matériel open source](https://oshwa.org/resources/open-source-hardware-definition/), du mobilier aux [FPGA](https://oshwa.org/resources/best-practices-for-sharing-fpga-designs/), sont couvertes par les licences CERN Open Hardware : [CERN-OHL-P-2.0](/licenses/cern-ohl-p-2.0/) (permissive), [CERN-OHL-W-2.0](/licenses/cern-ohl-w-2.0/) (faiblement réciproque) et [CERN-OHL-S-2.0](/licenses/cern-ohl-s-2.0/) (fortement réciproque).

### Projets mixtes

Si votre projet mêle des types de contenus différents, vous pouvez inclure plusieurs licences, à condition d’indiquer clairement quelle licence s’applique à chaque partie du projet. Voir [la notice de licence de ce site](https://github.com/github/choosealicense.com#license) en exemple.
